Description

Ce projet a été réalisé dans le cadre du module Introduction à l'Informatique L'objectif est de concevoir une application de calculatrice fonctionnelle tout en maîtrisant les bases du HTML, du CSS, du JavaScript, ainsi que les outils de déploiement modernes comme GitHub et Netlify
Fonctionnalités Réalisées Les boutons ont été transformés en cercles parfaits Un thème visuel cohérent a été appliqué avec des modifications sur les polices, les ombres et les arrondis Les boutons des opérateurs ont été déplacés sur le côté droit de la calculatrice pour une meilleure utilisation. Ajout de la fonction Nom de la fonction choisie : Puissance / Racine carrée
Documentation du Code Chaque ligne de code dans les fichiers index.html, style.css et script.js est commentée pour expliquer la logique et l'utilité des commandes utilisées.
